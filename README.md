# Korea Vibes Landing Page

Ruby Kiwi 제품을 소개하는 반응형 제품 소개 랜딩 페이지입니다.

## 프로젝트 주제
제품 소개 랜딩 페이지 만들기

## 제품
Ruby Kiwi

## 주요 기능
- 랜딩 페이지 기본 구조 구성
- 제품 이미지 배치
- 히어로 섹션 제작
- 제품 특징 카드 구성
- 내비게이션 바 제작
- 한국어 페이지와 영문 페이지 제작
- 모바일 및 태블릿 반응형 웹 적용
- 스크롤 등장 애니메이션 적용
- 제품 카드 플로팅 애니메이션 적용
- 버튼 및 카드 hover 애니메이션 적용

## 파일 구조
```txt
landing-page/
├─ index.html
├─ english.html
├─ style.css
├─ script.js
├─ ruby-kiwi.png
└─ README.md
```

## 사용 기술
- HTML5
- CSS3
- JavaScript
- Responsive Web Design

## 실행 방법
`index.html` 파일을 브라우저에서 실행하면 됩니다.
